NewsPrint API

NewsPrint allows you to save on the cloud your favorite web contents and convert them to a format suitable for printing or reading on your ebook reader. With NewsPrint you can also read your favorite RSS feeds in EPUB or PDF format.
Method Route Description Input Output
POST /register add a user to the database user data
POST /session create a new session user credentials user session id
POST /delete/*item_type*/*sessionid* delete an item object to delete
POST /submit/*sessionid* add a link/feed to list of links/feeds title and url of the item
POST /update/*sessionid* update last read date for a RSS feed url and UTC time
GET /links/*sessionid* get the list of web pages saved by the user a list of JSON Link objects
GET /feeds/*sessionid* get the list of RSS feed urls saved by the user a list of JSON Feed objects
